Tooltips

Like Popovers, only weak sauce.


Add tooltips to anything by adding the tooltip class and putting your content in a data-tooltip="My tooltip contents" attribute in that HTML element.

Change the tooltip positioning by adding the tooltip-right, tooltip-bottom or tooltip-left classes in addition to the tooltip class.

It's a menu/sidebar that lives offscreen and gets summoned at your command!

Sliders

Interactive sliders.


Add the tooltip class to add tooltip labels. If no value is specified by a data-tooltip="30" attribute, the value of the slider is displayed.


							<!-- Sliders -->
							<input class="slider" type="range" min="0" max="100" value="50">
							<!-- Sliders with tooltips -->
							<input class="slider tooltip" type="range" min="0" max="100" value="50" oninput="this.setAttribute('value', this.value);">
